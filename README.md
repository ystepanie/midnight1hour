# midnight1hour
헥사고날 아키텍처 구조를 사용하여 로그인 api 실습


## 🖥️ 프로젝트 소개
헥사고날 아키텍처 사용하여 Rest API 구현하였습니다.

GKE(Google Kubernetes Engine)를 사용하여 클라우드 배포 환경을 구축해보았습니다.

<br>

## 🕰️ 개발 기간
* 24.06.02일 - 24.07.01일


### ⚙️ 개발 환경
- `Java 21`
- **IDE** : InteliJ
- **Framework** : Springboot(3.x)
- **Database** : Maria DB
- **Database Tool** : Sequel Ace
- **ORM** : JPA
- **Build** : Gradle Groovy
- **Skills** : lombok, Docker, Kubernetes, GCP
